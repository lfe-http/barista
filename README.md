## Installation


Just add it to your ``rebar.config`` deps:

.. code:: erlang

    {deps, [
        ...
        {barista, ".*", {git, "git@github.com:lfex/barista.git", "master"}}
      ]}.


And then do the usual:

.. code:: bash

    $ rebar get-deps
    $ rebar compile
